# Superstore Sales Analysis

## Overview
This project analyzes Superstore sales data using SQL and Power BI to derive business insights related to sales performance, profitability, discounts, and regional trends.

## Tools Used
- MySQL
- Power BI
- VS Code

## Key Analysis
- Total Sales, Total Profit, Profit Margin
- Number of Unique Orders
- Year-wise Sales Trend
- Category & Sub-Category Performance
- Regional Profitability
- Impact of Discounts on Profit
- Average Shipping Delay

## Files
- `Superstore.sql` → SQL queries used for analysis
- `Superstore.pbix` → Interactive Power BI dashboard

## Dashboard Features
- KPI cards for Sales, Profit, Orders, Profit Margin
- Dynamic parameter-based filtering (Region, Category, Sub-Category, Year)
- Visual analysis of discounts vs profit and sales trends

## Outcome
This project demonstrates practical SQL querying skills, data cleaning, and business-focused dashboarding using Power BI.
